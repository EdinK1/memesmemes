# [MemesMemes](https://memesmemes.netlify.com/)

> Meme Generator built with React

---

## Installation

Clone Project

```shell
$ git clone https://github.com/EdinK1/memesmemes
```

Install Dependencies

```shell
$ npm i
```

Run Project

```shell
$ npm start
```
